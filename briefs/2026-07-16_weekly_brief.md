**2. Starbucks Is Building AI Software to Replace Microsoft, IBM, and Oracle**
- Using AI-assisted coding to build in-house replacements for Microsoft inventory mgmt, IBM maintenance software, and potentially Oracle's Simphony POS — $400M/year software spend is the target
- $30M tech savings this year ($10M lower software spend, $13M fewer contractors); part of $2B broader cost reduction; AI usage now a factor in performance reviews
- Caveat: simultaneously discontinued an AI inventory tracking system and returned to manual counting — execution is mixed
Source: [Storyboard18 / Bloomberg](https://www.storyboard18.com/brand-marketing/starbucks-taps-ai-to-build-in-house-software-reduce-reliance-on-microsoft-and-ibm-103772.htm)
